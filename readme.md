1. #### 项目背景

   该项目是针对QQ飞车手游的车队内道具赛尔而特别定制的选手匹配系统。(具体赛制待下版本详细说明)

   其中包含两个模块：

   * 选手等级划分

     根据本赛季段位、历史最高段位和本赛季胜率，对选手进行打分，根据得分进行聚类，划分为四个等级类别

   * 队伍生成

     给定一个选手需要轮到的场次数目，根据选手的等级和可选位置，随机抽取出两组水平相近的队伍

2. #### 上手指南

   * ##### 环境依赖

     * 语言

       Python 3.8.5

     * 第三方库
       
       * scikit-learn    0.23.2

   * ##### 运行

     * 随机生成选手数据

       `python  init_data_test.py`

     * 生成队伍

       `python match_system.py`
